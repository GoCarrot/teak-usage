= Markup Tags for Android Notification Text

HTML-lite tags are available in Android notification text.

== Supported Tags

The following tags are supported:

[horizontal]
``<b>``:: bold

``<i>``:: italic

``<tt>``:: monospace

``<u>``:: underline

``<strike>``:: strikethrough

``<font color="...">``:: change text color
